A simple plugin that fixes 404 error when product category base is set the same as shop base in WooCommerce

== Description ==

This plugin will allow you to create SEO friendly permalinks for your Woocommerce shop ( e.g. mydomain.com/category/my-category/ to mydomain.com/my-category/ ).

No configuration is required

mydomain.com/my-shop/my-category/my-sub-category/my-product/

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
